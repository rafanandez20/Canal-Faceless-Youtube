## PROMPT BASE (pegar siempre antes del guion)

Con estas imágenes de referencia quiero que mantengas EXACTAMENTE el mismo estilo visual para todas las imágenes que generemos en adelante.

IMPORTANTE:
- Todas las imágenes deben parecer hechas por el mismo autor.
- Mantén el mismo diseño de stickman en todo momento.
- No hagas el estilo más detallado con el tiempo.
- La simplicidad es OBLIGATORIA.

ESTILO VISUAL:
- Ultra-simple stickman cartoon
- Líneas negras gruesas
- Cabezas redondas blancas
- Ojos simples grandes
- Expresiones mínimas
- Brazos y piernas hechos con líneas simples
- Manos negras redondas
- Estilo educativo y humorístico
- Intencionalmente simple y ligeramente ridículo
- Aspecto de dibujo rápido hecho a mano
- Diseño fácil de animar

COLORES:
- Usar pocos colores
- Colores planos y suaves
- Fondos simples
- Evitar composiciones tipo cómic o viñetas
- NO dividir la pantalla en paneles
- NO hacer infografías complejas
- NO hacer mini escenas separadas

FONDOS:
- Muy minimalistas
- Solo añadir elementos necesarios
- Mucho espacio vacío
- Aspecto limpio
- El personaje siempre debe ser el foco principal

IMPORTANTE SOBRE EL TEXTO:
- TODO el texto debe estar SIEMPRE en español/castellano.
- Nunca uses inglés.
- Usa texto SOLO cuando tenga sentido dentro del entorno visual.

PERMITIDO:
- "Farmacia"
- "Ibuprofeno"
- "Salida"
- "Hospital"
- "WC"
- "Policía"

PROHIBIDO:
- escribir frases del guion
- subtítulos
- narración escrita
- captions
- texto explicativo
- párrafos
- títulos grandes
- bocadillos llenos de texto

MUY IMPORTANTE:
Las imágenes NO deben parecer miniaturas de YouTube ni cómics.
Deben parecer escenas simples de una animación stickman minimalista.

REGLA CLAVE:
Si una escena se entiende sin texto, NO pongas texto.

La comunicación visual debe venir de:
- poses
- expresiones
- objetos
- situación visual

NO uses:
- composiciones sobrecargadas
- exceso de objetos
- fondos detallados
- sombras complejas
- realismo
- estética Pixar
- anime
- estilo meme

OBJETIVO FINAL:
Crear escenas simples, limpias y visualmente coherentes para un canal faceless de antropología, ciencia y misterios humanos narrado con voz en off.

Guion a representar:
(AQUÍ PEGAS LA PARTE DEL GUIÓN)

Single clean scene, minimal composition, no comic panel layout, no infographic layout.
